# Sentiment Analysis using Machine Learning

## Description
This project predicts whether text feedback is Positive, Negative, or Neutral using Machine Learning and NLP techniques.

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NLP

## How it Works
- Text cleaning
- TF-IDF vectorization
- Naive Bayes model
- Sentiment prediction

## Example
Input: "This product is amazing"
Output: Positive
